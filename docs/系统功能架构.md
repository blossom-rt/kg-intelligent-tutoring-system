# KG 智能导学系统 — 功能架构

## 系统概述

面向大学生的知识图谱驱动个性化学习平台。以知识点为节点构建学科知识图谱，通过拓扑排序自动生成学习路径，配合在线刷题、测评、错题本和 AI 辅助，实现自适应学习闭环。

## 功能模块详解

### 1. 用户认证模块
- 三角色登录（学生/教师/管理员）
- 学生自主注册 + 邮箱验证码
- 密码重置
- JWT Token 鉴权 + 路由守卫

### 2. 知识图谱模块（核心引擎）
- **节点（knowledge_node）**：知识点实体，含名称/难度/章节/预计学时
- **边（knowledge_edge）**：前置依赖关系，形成有向无环图（DAG）
- ECharts 力导向图可视化展示
- 支持按课程筛选、拖拽缩放

### 3. 学习路径模块
- 选择目标知识点 → 拓扑排序自动生成学习路径
- 路径包含：前置依赖节点列表 + 学习顺序
- 支持按跨学科主题批量生成路径
- 进度追踪：实时显示已完成/总节点数

### 4. 在线刷题模块
- 按知识点或题目 ID 加载练习题
- 支持单选/多选/判断题
- 即时判分 + 正确答案展示 + 解析
- 答错自动收录错题本
- 学习伙伴（StudyPet）交互反馈

### 5. 测评模块
- 教师组卷：选择题目 + 设置总分
- 学生参加测评 → 自动阅卷
- AI 诊断报告（薄弱知识点识别）
- 测评记录查询

### 6. 错题本模块
- 答错自动收录，支持重做和 AI 讲解
- **当前错题** / **曾经错题** 双选项卡
- 重做答对自动移入"曾经错题"
- 错题本地存储（localStorage 辅助）

### 7. 跨学科主题模块
- 教师创建主题 → 关联多学科知识点
- 学生按主题浏览 → 一键生成跨学科学习路径
- 支持发布/下架状态管理

### 8. 学情分析模块
- 学生端：个人掌握度 / 正确率 / 薄弱知识点
- 教师端：班级整体分析 / 个体详情 / 薄弱知识点 TOP5
- 学习天数 / 时长 / 已掌握节点数 统计

### 9. AI 辅助模块
- AI 错题讲解（DeepSeek 接入）
- AI 学习路径总结
- AI 诊断报告生成
- AI 调用日志追踪

### 10. 教师管理模块
- 课程管理（CRUD）
- 知识点管理（CRUD + 难度/章节编排）
- 题库管理（CRUD + 知识点关联）
- 测评管理（组卷 + 阅卷）
- 图谱编辑（可视化节点/边操作）
- 学习路径督导（查看学生路径进度）

### 11. 管理员模块
- 用户管理 + 角色权限管理
- 系统公告发布
- 操作日志审计
- AI 调用日志监控

### 12. 设计系统
- 暖调浅色主题（`#faf7f2` + `#ff7b3d`）
- SVG 颗粒纹理全局覆盖
- 侧边栏导航（学生/教师/管理员）
- 学习伙伴宠物（StudyPet）
- 响应式布局（768px / 480px 断点）
- 暗黑模式支持

## 数据流

```
用户操作 → Vue前端 → Axios (/api/kg) → Vite代理 → SpringBoot (/api/*)
                                                    │
                                              Controller 层
                                                    │
                                              Service 层
                                                    │
                                              Mapper 层 (MyBatis-Plus)
                                                    │
                                              MySQL (19张表)
```

## 技术栈

| 层 | 技术 |
|---|---|
| 前端 | Vue 3 + Vite + Element Plus + ECharts + Pinia |
| 后端 | Spring Boot 3.5 + MyBatis-Plus + JWT |
| 数据库 | MySQL 8.0（19 张表） |
| AI | DeepSeek API |
| 部署 | Docker Compose |
